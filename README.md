# Inventory Strategic & Risk Management Dashboard 📊

## 📌 Project Overview
This project focuses on optimizing inventory management and mitigating financial risks associated with stock expiration and disposal delays. Using  **Excel**, I developed a strategic dashboard to provide a real-time overview of stock health, supplier performance, and warehouse efficiency.

---

## 📊 Key Insights & Business Value
* **Inventory Valuation:** Monitored and managed a total stock value of **$333K**.
* **Risk Mitigation:** Tracked **$171K** in expired goods and identified **$4K** worth of "near-to-expire" stock to support proactive decision-making.
* **Warehouse Analysis:** Analyzed supplier loss risk and disposal delays, identifying top high-risk warehouses (e.g., **8851 Superior Park**).
* **Stock Optimization:** Categorized stock status across multiple segments (Dairy, Fruits, Beverages) to optimize replenishment cycles and minimize loss.

---

## 🛠️ Tech Stack
* **Analysis:** Microsoft Power BI (DAX, Data Modeling).
* **Data Prep:** Microsoft Excel (Power Query).
* **Concepts:** Inventory Turnover, Risk Assessment, Trend Analysis.
